## Installing Arch Linux

### Requirements

-   usb stick (min 8gb)
-   stable internet connection
-   coffee

> This is an important note: follow official docs and checksum the ISO before flashing.

Go to the [Arch Linux website](https://archlinux.org) and download the latest ISO. Choose a nearby mirror to avoid download issues. On Windows you can use Rufus; on Linux use `dd` or `ventoy`.

### Quick install steps (high level)

1. Create a bootable USB from the ISO.
2. Boot installer and connect to the internet.
3. Partition, format, and install base packages.
4. Reboot and continue system configuration.

fn get_median(list: &mut [i32]) -> f32 {
    let len = list.len();
    let mid = len / 2;

    list.sort();

    if len.is_multiple_of(2) {
        (list[mid - 1] as f32 + list[mid] as f32) / 2.00
    } else {
        list[mid] as f32
    }
}

fn get_mode(list: &[i32]) -> i32 {
    let mut freq = HashMap::new();
    let mut max_value = 0;
    let mut max_key = 0;

    for num in list {
        *freq.entry(num).or_insert(0) += 1;
    }

    for (key, value) in freq {
        if value > max_value {
            max_key = *key;
            max_value = value;
        }
    }

    max_key
